
## Install

Using Composer

```
composer require ken/spider-admin
```

Add the service provider to `config/app.php`

```php
Ken\SpiderAdmin\App\Providers\SpiderAdminServiceProvider::class,
```
and
```
php artisan vendor:publish
```
## Usage

### Basic

Route Basic

localhost:8000/spider
